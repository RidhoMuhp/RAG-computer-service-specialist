# Laptop Layar Hitam (Black Screen Issue)

## Problem Category
Display

## Affected Devices
- Laptop (All Brands)

## Common Symptoms
- Laptop menyala tetapi layar tetap hitam
- Lampu indikator hidup namun tidak ada tampilan
- Kipas berputar tetapi tidak ada gambar
- Layar hitam setelah login atau saat booting
- Tampilan hanya terlihat saat menggunakan monitor eksternal
- Layar berkedip lalu menjadi hitam

## Possible Root Causes
- Kabel fleksibel LCD longgar atau rusak
- Panel LCD rusak atau mati
- Driver VGA bermasalah atau corrupt
- IC grafis (GPU) bermasalah
- IC backlight atau inverter rusak
- Tegangan ke layar tidak stabil
- BIOS error atau gagal update
- RAM bermasalah atau tidak terbaca

## Initial Diagnostic Steps
1. Hubungkan laptop ke monitor eksternal
2. Cek apakah layar menampilkan gambar dengan senter (backlight test)
3. Reset RAM dan bersihkan slot RAM
4. Masuk ke BIOS untuk memastikan tampilan dasar muncul
5. Boot menggunakan safe mode untuk cek driver VGA

## Recommended Solutions
- Perbaiki atau ganti kabel fleksibel LCD
- Ganti panel LCD jika tidak menampilkan gambar
- Install ulang atau update driver VGA
- Reset atau update BIOS
- Reball atau perbaiki IC GPU (level lanjutan)
- Ganti RAM jika terdeteksi bermasalah

## Cost Estimation (IDR)
- Diagnosa: Rp 50.000 – Rp 150.000
- Ganti kabel fleksibel: Rp 150.000 – Rp 400.000
- Ganti LCD panel: Rp 600.000 – Rp 2.500.000
- Perbaikan GPU / IC display: Rp 800.000 – Rp 3.000.000

## Time Estimation
- Diagnosa: 30 – 60 menit
- Penggantian kabel LCD: 60 – 120 menit
- Penggantian LCD panel: 60 – 180 menit

## Risks & Warnings
- Membuka layar berisiko merusak panel jika tidak hati-hati
- Salah penanganan GPU dapat menyebabkan kerusakan motherboard
- Update BIOS yang gagal dapat menyebabkan laptop mati total

## Additional Notes
- Gunakan monitor eksternal untuk membedakan masalah layar atau motherboard
- Hindari menekan layar secara berlebihan
- Pastikan menggunakan driver VGA resmi dari pabrikan
